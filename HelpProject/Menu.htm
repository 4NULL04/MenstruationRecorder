You can do some basic operations to the data and the software.

  • New - create a new record. You can use accelerator of Ctri + N.
  • Open - open a existing record. You can use accelerator of Ctrl + O.
  • Save - save current data. You can use accelerator of Ctrl + S.
  • Save As - save current data in a different record.
  • Exit - exit the software with automatic saving.
